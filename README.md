# Netflix Exploratory Data Analysis

## Problem Statement :
The objective of this project is to analyze the Netflix Movies and TV Shows dataset and uncover meaningful insights using Exploratory Data Analysis (EDA) techniques.

## Dataset Details :
- Dataset: Netflix Movies and TV Shows
- Source: Kaggle
- Total Records: 8,807
- Features(12): Show ID, Type, Title, Director, Cast, Country, Date Added, Release Year, Rating, Duration, Genre, Description

## Approach :
1. Loaded and explored the dataset using Pandas.
2. Performed data cleaning by handling missing values and checking duplicates.
3. Conducted exploratory data analysis using statistical summaries and visualizations.
4. Analyzed trends related to content type, countries, ratings, genres, directors, actors, and release years.
5. Performed correlation analysis using a heatmap.

## Results :
- Movies dominate Netflix's content catalog.
- Netflix experienced significant growth in content additions over the years.
- The United States contributes the highest number of titles.
- TV-MA and TV-14 are among the most common ratings.
- Drama and International Movies are among the most popular genres.
- Most content originates from recent years.
- Certain directors and actors appear frequently across Netflix titles.
